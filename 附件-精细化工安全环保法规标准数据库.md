```yaml
编号: GB 45673-2025
名称: 危险化学品企业安全生产标准化通用规范
实施日期: 2025-11-01
性质: 强制性国家标准
替代标准: AQ3013-2008等7项行业标准
适用对象: 精细化工企业、医药中间体生产企业

核心章节:
  4.2 基本要求:
    - 高危工艺全流程自动化要求
    - 硝化/氯化/氟化/重氮化/过氧化工艺
    - 2024年底硝化完成；2026年底其他完成
    - 危险度3级及以上：微通道/管式反应器替代
    
  5.1 安全领导力:
    - 主要负责人组织实施安全生产信息化智能化建设
    - 2024年底前重大危险源企业建成：
      * 特殊作业许可与作业管理系统
      * 人员定位系统（含人员聚集风险监测预警）
    - 2025年开展报警优化管理、关键机泵状态监测、承包商管理
    
  5.5 安全风险管理和双重预防机制:
    - 双重预防数字化系统与巡检系统融合
    - 高后果生产装置、储存设施重点管控
    - 重大危险源三类包保责任人履职
    
  5.6 设备完整性（新增）:
    - 设备全生命周期管理
    - 二级要素:
      * 5.6.1 采购与安装
      * 5.6.2 分级管理（按重要性、故障后果）
      * 5.6.3 检验检测和检查（腐蚀评估、在线监测）
      * 5.6.4 预防性维护和维修
      * 5.6.5 检维修质量控制
      * 5.6.6 缺陷管理
      * 5.6.7 安全设施（SIS完整性等级评估）
      * 5.6.8 特种设备
      * 5.6.9 设备完整性数据库（打破信息孤岛）
      * 5.6.10 证实方法
      
  5.8 作业安全:
    - 2024年典型事故：10起中6起涉及动火、受限空间
    - 重大危险源企业须建设应用特殊作业审批与管理场景功能
    
  5.10 化学品安全和重大危险源:
    - 涉及"两重点一重大"装置应开展HAZOP分析
    - 重大危险源按GB 17681-2024设置SIS等安全监控系统
    - 分析化验室安全：通风、试剂分类存放、报警消防设施

检查要点:
  - [ ] 是否完成高危工艺全流程自动化改造评估
  - [ ] 是否建立设备完整性数据库
  - [ ] 是否建设特殊作业许可与管理系统
  - [ ] 是否配备人员定位系统
  - [ ] 重大危险源三类包保责任人是否明确
  - [ ] 是否开展年度HAZOP分析

相关文件:
```

#### 2. GB/T 42300-2022 精细化工反应安全风险评估规范
```yaml
编号: GB/T 42300-2022
名称: 精细化工反应安全风险评估规范
实施日期: 已实施
性质: 推荐性国家标准
适用对象: 精细化工反应安全风险评估

核心内容:
  评估方法:
    - 单因素评估：反应热、绝热温升、最大反应速率到达时间(TMRad)
    - 混合叠加因素评估：严重度+可能性矩阵
    - 反应工艺危险度评估：四个温度参数排序
    
  关键参数:
    Tp: 工艺温度
    MTSR: 失控体系能达到的最高温度
    MTT: 技术最高温度（沸点或最大允许压力对应温度）
    TD24: 最大反应速率到达时间为24小时对应的温度
    ΔTad: 绝热温升
    TMRad: 最大反应速率到达时间（致爆时间）
    
  危险度分级:
    1级: Tp < MTSR < MTT < TD24 | 危险性较低 | 常规DCS/PLC
    2级: Tp < MTSR < TD24 < MTT | 潜在分解风险 | 报警联锁+泄放+SIS
    3级: Tp ≤ MTT < MTSR < TD24 | 冲料和分解风险 | 紧急冷却/减压+新技术替代
    4级: Tp ≤ MTT < TD24 < MTSR | 冲料和分解风险高 | 工艺优化+隔离措施
    5级: Tp < TD24 < MTSR < MTT | 爆炸风险高 | 工艺路线改变+区域隔离
    
  严重度评估(绝热温升):
    1级: ΔTad ≤ 50K | 单批次物料损失
    2级: 50 < ΔTad < 200K | 工厂短期破坏
    3级: 200 ≤ ΔTad < 400K | 工厂严重损失
    4级: ΔTad ≥ 400K | 工厂毁灭性损失
    
  可能性评估(TMRad):
    1级: ≥24h | 很少发生
    2级: 8-24h | 偶尔发生
    3级: 1-8h | 很可能发生
    4级: ≤1h | 频繁发生

检查要点:
  - [ ] 高危工艺是否完成全流程反应安全风险评估
  - [ ] 评估报告是否包含热力学和动力学测试
  - [ ] 工艺危险度3级及以上是否制定替代方案
  - [ ] 是否按评估建议落实控制措施
  - [ ] 是否建立化学品相容性矩阵
```

#### 3. AQ 3062-2025 精细化工企业安全管理规范
```yaml
编号: AQ 3062-2025
名称: 精细化工企业安全管理规范
实施日期: 即将实施
性质: 安全生产行业标准
适用对象: 精细化工企业

核心章节:
  4.2 本质安全原则:
    - 最小化、替代、缓和、简化
    - 优先选用：
      * 微通道反应、管式反应、环流反应等连续化技术
      * 低危险性化学品替代高危险性化学品
      * 温和反应条件（温度、压力）
      * 步骤少、易自动化的工艺
      
  5.2.3 反应安全风险评估:
    - 高危工艺必须完成全流程评估
    - 全流程范围：原料预处理→分步化学反应→产品分离精制
    
  5.2.4 危险度3级及以上要求:
    - 优先采用微通道、管式反应器等新装备
    - 暂不具备条件的：反应器置于防爆建构筑物内+安全可靠性论证
    
  5.2.5 过程危险性分析:
    - 基础工程设计阶段采用HAZOP、LOPA方法
    - 涵盖：操作偏差、加料失控、搅拌突停、冷媒断供、突然停电
    - 包含：活化、加料、反应、分离、退料、干燥、清洗、输送、储存
    
附录A 典型危险化工工艺安全风险管控关键要求:
  A.1 硝化工艺:
    - 危险度3级及以上：必须采用微通道/管式反应器+全流程自动化
    - 连续化工艺定期检测硝化物、副产物浓度
    - 重点监控：反应温度、搅拌电流、硝化剂流量、冷媒压力、冷却水pH
    - 反应器至少2只温度计，定期校验
    - 限流措施：限制进料管径、限流孔板、调节阀物理限位
    - 停车时双切断措施
    - 加热介质温度超过TD24时：紧急冷却系统
    - 惰性气体保护+吹扫设施
    - 远程视频监控+声光报警
    - 减缓措施：抑制淬灭、骤冷浇灌、倾泻排放、安全泄放、物料置换
    
  A.2 氯化/氟化/重氮化/过氧化工艺:
    - 参照硝化工艺执行

检查要点:
  - [ ] 建设项目是否有明确工艺技术来源
  - [ ] 是否建立化学品相容性矩阵
  - [ ] 是否开展HAZOP分析
  - [ ] 危险度3级及以上是否制定新技术替代方案
```

#### 4. GB 51283-2020 精细化工企业工程设计防火标准
```yaml
编号: GB 51283-2020
名称: 精细化工企业工程设计防火标准
实施日期: 已实施
性质: 强制性国家标准
适用对象: 精细化工企业工程设计

核心章节:
  4.2 防火间距:
    - 与居住区、村镇：90-140m
    - 与相邻工厂：27-40m
    - 与国家铁路：60-70m
    - 与高速公路：25m
    - 与变配电所：30-55m
    
  5.5 生产设施内布置:
    车间储罐（组）要求:
      - 储量≤1天需求量
      - 可燃气体总容积≤1000m³
      - 液化烃总容积≤100m³
      - 可燃液体总容积≤1000m³
      - 不得布置在封闭式/半敞开式厂房内
      
    设备布置防火间距:
      - 操作温度≥自燃点的设备与甲类气体设备：≥4.5m
      - 与液化烃设备：≥7.5m
      
  8.3 厂（库）房平面布置:
    - 甲、乙类厂房：办公室/休息室/控制室/化验室不应设置在内
    - 确需贴邻：耐火极限≥3.00h防爆墙+独立安全出口
    - 丙类厂房：耐火极限≥2.50h防火隔墙+1.00h楼板分隔
    - 变配电所：不应在甲、乙类厂房内或贴邻
    - 20kV及以下专用变配电所：可无门窗洞口防火墙一面或两面贴邻

检查要点:
  - [ ] 总平面布置防火间距是否符合要求
  - [ ] 车间储罐储量是否超标
  - [ ] 甲/乙类厂房是否违规设置办公休息区
  - [ ] 变配电所设置是否符合要求
```

#### 6. GB 37823-2019 制药工业大气污染物排放标准
```yaml
编号: GB 37823-2019
名称: 制药工业大气污染物排放标准
实施日期: 已实施
性质: 强制性国家标准
适用对象: 制药工业企业（含医药中间体参照执行）

排放限值:
  有组织排放:
    NMHC: ≤60mg/m³
    TVOC: ≤100mg/m³
    苯系物: ≤40mg/m³
    光气: ≤0.5mg/m³
    氯化氢: ≤30mg/m³
    氨: ≤20mg/m³
    硫化氢: ≤5mg/m³
    臭气浓度: ≤1000（无量纲）
    
  无组织排放:
    NMHC（厂房外监控点）: ≤6mg/m³（1h平均）
    
  特别排放限值（重点地区）:
    NMHC: ≤50mg/m³
    TVOC: ≤80mg/m³

控制要求:
  - 废气应分类收集、分质处理
  - 含药物活性成分废气应净化处理
  - 发酵尾气应除臭处理
  - 有机溶剂废气应回收
  - 废水处理设施废气应收集处理
  
检查要点:
  - [ ] 排气筒污染物是否达标排放
  - [ ] 厂区内VOCs无组织排放是否达标
  - [ ] 是否建立VOCs治理设施运行台账
  - [ ] 是否定期开展LDAR泄漏检测与修复
```

#### 7. GB 37822-2019 挥发性有机物无组织排放控制标准
```yaml
编号: GB 37822-2019
名称: 挥发性有机物无组织排放控制标准
实施日期: 已实施
性质: 强制性国家标准
适用对象: 涉及VOCs物料的工业企业

控制环节:
  VOCs物料储存:
    - 储罐：压力罐、内浮顶罐、固定顶罐+密闭收集
    - 储罐呼吸阀定期检测
    
  VOCs物料转移和输送:
    - 密闭管道输送
    - 汽车/火车装载：底部装载+气相平衡系统
    
  工艺过程:
    - 投料、反应、萃取、蒸馏等应密闭
    - 真空泵排气应收集处理
    - 设备清洗应收集VOCs
    
  敞开液面:
    - 废水集输系统：密闭管道/沟渠
    - 废水储存设施：固定顶罐+密闭收集+处理设施
    
  泄漏检测与修复(LDAR):
    - 泵、压缩机、阀门、开口阀、法兰等密封点
    - 检测频次：根据泄漏认定浓度和组件类型
    - 修复时限：首次/延期

厂区内VOCs无组织排放限值:
  监控点处1h平均浓度: NMHC ≤6mg/m³
  监控点处任意一次浓度: NMHC ≤20mg/m³

检查要点:
  - [ ] 储罐类型是否符合要求
  - [ ] 物料转移是否密闭
  - [ ] 是否建立LDAR制度并实施
  - [ ] 废水处理设施废气是否收集处理
  - [ ] 厂区内VOCs监测是否达标
```

#### 8. GB 18597-2023 危险废物贮存污染控制标准
```yaml
编号: GB 18597-2023
名称: 危险废物贮存污染控制标准
实施日期: 已实施
性质: 强制性国家标准
替代: GB 18597-2001

核心要求:
  贮存设施选址:
    - 符合国土空间规划
    - 避开活动断层、溶洞区、天然滑坡区
    - 避免建在江河、湖泊、运河等最高水位线以下
    
  贮存设施污染控制:
    - 地面防渗：渗透系数≤1×10⁻¹⁰cm/s，或2mm厚HDPE膜
    - 渗滤液收集系统
    - 气体收集净化系统（易产生VOCs、有毒有害气体的危废）
    - 贮存设施内清理出的泄漏物按危废管理
    
  容器和包装物:
    - 材质、内衬与危废相容
    - 破损及时更换
    
  贮存过程:
    - 危废分类分区贮存
    - 不相容危废分开存放
    - 容器标识：标签+警示标志
    
  贮存设施运行:
    - 24小时视频监控
    - 电子管理台账
    - 定期检查包装容器、防渗漏措施、标签标识
    - 危废贮存期限≤1年（需延长的报批）

检查要点:
  - [ ] 危废库防渗是否符合要求
  - [ ] 是否安装24小时视频监控
  - [ ] 是否建立电子管理台账
  - [ ] 危废标识标签是否规范
  - [ ] 是否存在超期贮存
  - [ ] 是否定期转移处置
```

### 安委〔2024〕2号 化工和危险化学品安全生产治本攻坚三年行动方案（2024-2026年）
```yaml
文件: 化工和危险化学品安全生产治本攻坚三年行动方案（2024-2026年）
文号: 安委〔2024〕2号
发布: 2024年

核心任务:
  高危工艺全流程自动化改造:
```
