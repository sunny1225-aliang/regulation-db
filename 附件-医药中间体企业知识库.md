## 二、行业动态库

### 2.1 技术路线趋势

#### 抗肿瘤中间体
- **细胞毒性药物（HPAPI）**：密闭隔离系统（Isolator/RABS）成为标配
- **OEL/OEB评估**：职业暴露分级管理趋严
- **连续流化学**：微通道反应器替代传统釜式反应（政策推动+安全需求）

#### 核酸类（寡核苷酸/siRNA）
- **固相合成**：主流工艺，关注杂质谱控制
- **监管框架**：CDE技术指导原则持续更新
- **CDMO机会**：创新药企外包需求增长

#### 保健品
- **功能声称**：监管趋严，需科学证据支持
- **原料目录**：动态调整，关注新原料备案
- **注册/备案双轨**：理解适用边界

## 三、管理方法论库

### 3.1 生产管理

#### 设备完整性管理（GB 45673-2025新要求）
```
全生命周期节点：
采购与验收 → 分级管理 → 检验检测 → 预防性维护 → 检维修质量控制 → 缺陷管理 → 数据库
```

**关键动作**：
1. 建立设备完整性数据库（打破信息孤岛）
2. 制定设备分级标准（重要性+故障后果）
3. 腐蚀评估与在线监测（尤其是涉及强酸强碱设备）

#### 双重预防机制
- 风险分级管控 + 隐患排查治理
- 数字化系统与巡检系统融合
- 重点关注：重大危险源三类包保责任人履职

### 3.2 质量管理

#### 变更管理
| 变更类型 | 管理要求 |
|---------|----------|
| 工艺变更 | 风险评估→验证→审批→培训→归档 |
| 设备变更 | 确认（IQ/OQ/PQ）→ 工艺验证 |
| 供应商变更 | 审计→样品测试→小试→中试 |

#### 文件体系
- 污染控制策略（CCS）文件整合
- 数据完整性（ALCOA+原则）
- 批生产记录与SOP同步更新
