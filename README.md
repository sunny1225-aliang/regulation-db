# 精细化工安全环保法规库 - 网页版

> 纯浏览器运行，无需安装，数据本地存储

---

## 🚀 快速开始

### 方法一：直接打开（最简单）
1. 下载整个 `web` 文件夹
2. 双击 `index.html`
3. 浏览器自动打开，即可使用

### 方法二：部署到服务器（推荐）
把 `web` 文件夹上传到任意 Web 服务器：
- **GitHub Pages**（免费）
- **Cloudflare Pages**（免费，国内快）
- **Vercel / Netlify**（免费）
- **自己的服务器/NAS**

### 方法三：手机添加到桌面（PWA）
1. 用 Safari（iOS）或 Chrome（Android）打开网页
2. 点击"分享" → "添加到主屏幕"
3. 像原生 App 一样使用，支持离线访问

## 📱 使用说明

### 快捷键

| 按键 | 功能 |
|------|------|
| `/` | 快速聚焦搜索框 |
| `ESC` | 关闭弹窗/侧边栏 |
| `1-5` | 切换导航（需先打开菜单） |

### 数据存储说明

**所有数据保存在浏览器本地**：
- ✅ 收藏夹列表
- ✅ 检查清单完成状态
- ✅ 阅读记录（将来版本）

**数据不会丢失，除非**：
- 清除浏览器缓存/数据
- 使用无痕模式浏览
- 更换设备或浏览器

**备份方法**：
```javascript
// 在浏览器控制台执行
const fav = localStorage.getItem('regulation_favorites');
const chk = localStorage.getItem('regulation_checklist');
console.log({favorites: fav, checklist: chk});
// 保存输出的内容
```

## 🔄 数据更新

### 自动更新（未来支持）
部署到服务器后，修改 `data.js` 即可自动更新所有用户看到的内容。

### 手动更新
1. 编辑 `data.js` 文件
2. 添加/修改法规标准
3. 刷新浏览器即可

### 数据格式
```javascript
// 添加新标准
{
    id: "唯一标识",
    code: "标准编号",
    name: "标准名称", 
    category: "安全/环保",
    subcategory: "子类别",
    type: "强制性国标/推荐性国标/行业标准",
    effectiveDate: "实施日期",
    status: "现行有效/即将实施",
    keywords: ["关键词"],
    summary: "标准摘要",
    keyPoints: ["要点1", "要点2"],
    checkItems: [
        { item: "检查项", required: true, deadline: "截止时间" }
    ]
}
```

## 💡 使用技巧

1. **搜索技巧**：输入"GB 45673"或"设备完整性"或"微通道"都能搜到
